This project demonstrates a Text-to-Image Generation application built using Python, Diffusion-based models, and Gradio.
Users can enter a text prompt, and the model generates an image based on the given description through an interactive web interface.


Features :-
Convert natural language text prompts into image
Simple and interactive Gradio web interface
Runs seamlessly inside Jupyter Notebook
Beginner-friendly and easy to extend
Uses modern AI image generation techniques

Future Improvements
Add multiple image generation
Prompt history & download option
Deploy as a public web app
Model fine-tuning
Support for styles (anime, realistic, oil painting)
